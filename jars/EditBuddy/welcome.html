Install some plugins to make jEdit even more useful to you.

To download and install plugins, make sure you are connected to the Internet, select Plugins>Plugin Manager from the menu bar, and click Install New Plugins in the dialog box that appears.

If you are connected to the Internet from behind a firewall, you will need to configure firewall settings before plugin installation will function.

Firewall settings can be configured in the Plugin Options>Firewall pane of the Utilities>Global Options dialog box.

To keep up-to-date in the world of jEdit, subscribe to the jEdit mailing lists.

Three lists have been set up; one for announcements only, one for general discussion, and one for technical discussion. See http://jedit.sourceforge.net for details.
