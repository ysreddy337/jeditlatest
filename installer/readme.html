This installer will install the jEdit programmer's text editor on your computer.

How to use this installer
Simply fill out the required information at each screen and click Next. Once all installation parameters have been specified, the Next button changes to read Install. Clicking it will then begin the installation process.

The Cancel button will quit the installer without installing anything; Previous will take you to the previous screen (use it if you change your mind about a setting).

System requirements
jEdit requires either Java 1.1 with Swing 1.1 or Java 2 to work. If this installer started without any errors, don't panic; you already have everything required to install jEdit.
